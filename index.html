About Us

Cryptic Languages

What comes to mind when you hear the word ‘coding’? Is it lines and lines of incomprehensible gibberish that looks like this? Well, learning to code is one of the best ways to land a good job. Did you know that just 8% of all STEM degrees are in computer science, while 71% of all STEM occupations are computing ones? There’s a lot of growth in the tech industry, and you should cash in on all that opportunity by learning to code. If you’re wondering where to learn to code, Cryptic Languages is just the place to start! Cryptic Languages is an online platform that’s open to all for learning to code. Currently, there are video tutorials for HTML and CSS on the site, with tutorials for more languages coming soon. Using HTML and CSS, you can create beautiful websites from scratch, all with your own work. Cryptic Languages is a place for beginners to learn to code in an easy and comfortable manner.

I founded this company because I realized that there are so many kids that don’t know how to code. Coding is an essential skill everyone needs to know, especially as the world becomes more integrated with tech. I myself didn’t learn to code until last year, but it honestly changed the trajectory of my life. I went from wanting to be a climate change analyst to a developer at Google in the span of one day, that’s how deep the chord of coding rang in me. I also wanted to create a free learning platform that people of all economic backgrounds can access. I didn’t want a child to miss out on the opportunity of learning to code just because of financial barriers! So many opportunities open up to people that know how to code, and it’s a great skill to have for the twenty first century. So, what are you waiting for? Check out the tutorials below to learn to code with Cryptic Languages!

Web Design

HTML

HTML is one of the three main languages used for web design. It's used to build the foundation of the website, with mainly just the text, usually it isn't used to add design.

CSS

CSS is one of the other main languages used for web design. CSS is used to design the website, to style the way the text is layed out.
